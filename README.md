# Contador de Letras e Palavras

Este é um projeto simples de um contador de caracteres e palavras, desenvolvido com HTML, CSS e JavaScript. Ele permite que o usuário insira um texto em uma área de texto e escolha entre contar o número de caracteres ou palavras no texto.

## Funcionalidades

- Contar o número de caracteres em um texto.
- Contar o número de palavras em um texto.
- Alternar entre os modos de contagem (caracteres ou palavras) com um botão.

## Estrutura do Projeto

O projeto é composto pelos seguintes arquivos:

- **index.html**: Contém a estrutura HTML da aplicação.
- **style.css**: Contém os estilos para a interface do usuário.
- **script.js**: Contém a lógica do contador de caracteres e palavras.

## Como usar

1. Clone este repositório para o seu computador.
2. Abra o arquivo `index.html` em um navegador web.
3. Digite ou cole o texto na área de texto exibida na página.
4. Clique no botão para alternar entre os modos de contagem de caracteres e palavras.

## Estrutura do Código

### HTML
O arquivo `index.html` define a estrutura básica da aplicação, incluindo:
- Um título.
- Uma área de texto para entrada do usuário.
- Um botão para alternar entre os modos de contagem.
- Um contador que exibe o número de caracteres ou palavras.

### CSS
O arquivo `style.css` estiliza a aplicação, fornecendo:
- Layout centralizado.
- Estilo para a área de texto e o botão.
- Efeitos de hover no botão.

### JavaScript
O arquivo `script.js` implementa a lógica da aplicação:
- Escuta eventos de entrada na área de texto para atualizar o contador em tempo real.
- Alterna entre os modos de contagem de caracteres e palavras ao clicar no botão.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
